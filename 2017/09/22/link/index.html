用于创建 符号链接 或 硬链接。

###查看

ls -l 可以查看文件是否为链接文件：

l 代表文件是链接文件；
netcat 是指向 /etc/alternatives/netcat 的一个文件链接。

###创建

  • ln -s target link 创建一个名为 link 的文件，链接到 target 文件；
  • ln -s target 在当前目录创建一个链接到 target 的文件，与 target 同名；
  • ln -s targets link_dir 在 link_dir 目录创建所有 targets 的链接。
