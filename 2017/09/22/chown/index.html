chown

Posted on 2017-09-22

修改文件或目录的所有者或（和）用户组。

###查看归属信息

ls -l 命令可以看到文件的拥有者和用户组：

root owner;
hu group.

###修改归属

  • sudo chown hu hello.js 将 hello.js 的 owner 改为 hu；
  • sudo chown root:root hello.js 将 hello.js 的 owner 改为 root，group 改为 root。

注意：chown 命令只能被 root 用户执行。
