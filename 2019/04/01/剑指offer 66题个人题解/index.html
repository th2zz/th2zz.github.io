1. 二维数组查找

在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。leetcode 74
方法一：暴力枚举 row major order 遍历 O(mn)效率很差
方法二：从右上角开始step search每一步比当前元素小就往左走 比当前元素大就往下走
step search on sorted matrix worst case O(m+n) 需要行内严格排序 行与行之间可以不排序
例如

public class Solution {
public boolean Find(int target, int [][] array) {
if(array == null || array.length == 0 || array[0].length == 0) return false;
for (int row = 0, col = array[0].length - 1; row <= array[0].length - 1 && col >= 0;) {
if (array[row][col] == target) return true;
else if (array[row][col] < target) row++;
else col--;
}
return false;
}
}

特殊情况：若矩阵严格row major order排序，则可以使用binary search O(lg(m+n))
例如
[1, 3, 5, 7]
[10, 11, 16, 20]
[23, 30, 34, 50]
binary search on 2d fully sorted row major order matrix

class Solution {
public boolean searchMatrix(int[][] matrix, int target) {
if(matrix.length == 0 || matrix[0].length == 0) return false;
int n = matrix.length, m = matrix[0].length;
int l = 0, r = m * n - 1;
while (l != r){
int mid = (l + r - 1) >>> 1;
if (matrix[mid / m][mid % m] < target) l = mid + 1;
else r = mid;
}
return matrix[r / m][r % m] == target;
}
}

2. 替换空格

请实现一个函数，将一个字符串中的每个空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy
方法一：直接遍历替换，每一次遇到空格替换为%20并将后续字符串后移2位 O(N^2)
方法二：O(N) TIME O(N) SPACE WORST CASE (All space多分配两倍内存)

public static String replaceSpace(StringBuffer str) {
if (str == null || str.length() == 0) return "";
int num_space = 0;
//提前数一下空格数量 
for(int i = 0; i < str.length();i++) if(str.charAt(i) == ' ') num_space++;
//if (num_space == 0) return str.toString();
int i = str.length() - 1, j = i + 2*num_space;//i指向原str末尾 j指向新str末尾 1 space induces 2 addtional units of storage
str.setLength(j+1);//预分配j+1个单位的的内存（因为j是index）
for (;i>=0 && j>=0;i--) {//loop backwards
if(str.charAt(i)==' ') {//当前字符若为空格 倒着连续替换3个字符 decrement j afterwards
str.setCharAt(j--,'0');
str.setCharAt(j--, '2');
str.setCharAt(j--, '%');
} else str.setCharAt(j--, str.charAt(i));//otherwise, shift what i points to to pos j;decrement j afterwards
}
return str.toString();
}

3. 从尾到头打印链表

输入一个链表，按链表值从尾到头的顺序返回一个ArrayList。 leetcode 206
方法一：反转链表之后把值遍历加进arraylist O(N) TIME O(1) SPACE（如果可以inplace modify linkedlist）

import java.util.ArrayList;
public class Solution {
public ArrayList<Integer> printListFromTailToHead(ListNode head) {
ArrayList<Integer> result = new ArrayList<Integer>();
ListNode curr = head, prev = null;
while (curr != null) {
ListNode temp = curr.next;
curr.next = prev;
prev = curr;
curr = temp;
}
while (prev != null) {
result.add(prev.val);
prev = prev.next;
}
return result;
}
}

方法二：使用stack来反转值的顺序 因为stack是顺序储存先进先出 可以起到反转顺序的作用 O(N) TIME O(N) SPACE

public ArrayList<Integer> printListFromTailToHead(ListNode head) {
ArrayList<Integer> result = new ArrayList<Integer>();
Stack<ListNode> s = new Stack<ListNode>();
ListNode curr = head;
for(;curr != null;curr = curr.next) s.push(curr);
for(;!s.isEmpty();) result.add(s.pop().val);
return result;
}

4. 重建二叉树

输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

首先我们知道preorder[0]是root 通过找到他在inorder中的位置我们可以知道左子树右子树的节点数量和具体节点
对于每一个子树的节点列表 preorder’’[0]仍是子树的根这样递归的进行树的重建。 但是对于非平衡树 或者极端情况一条链表 在inorder里搜索跟的位置要时间复杂度O(n^2) 并且取决于怎样split我们甚至可能需要空间复杂度O(n^2) 我们可以使用map来降低时间空间复杂度 但仍要O(n)
若不使用map我们也可以达到时间O(n)空间O(n)我们手动设置递归终止条件：
每个stack frame上我们使用当前子树根节点的值作为其左子树的递归终止条件 右子树使用当前的stop值作为递归终止条件
Instead of finding a subtree root in inorder, splitting the arrays into parts and recursing on them, just recurse on the full remaining arrays and stop when you come across the subtree root in inorder. Each recursive call gets told where to stop, and it tells its subcalls where to stop. It gives its own root value as stopper to its left subcall and its parent‘s stopper as stopper to its right subcall.
preorder = [3,9,20,15,7]
inorder = [9,3,15,20,7]
递归树：

                                     [3 stop=MIN]
/            \
/              \
/                \
/                  \
[9 stop=3]                   [20 stop=MIN]
/         \                     |    \        
/           \                    |     \
[null:inorder[0]=stop=9][null:inorder[1]=stop=3]      |      \
|       \
[15 stop=20]   [7 stop=MIN]
/      \                | \
/        \               |  \
/          \              |   \
/            \             |    \
/              \            |     \
[null:inorder[2]=stop=15][null:inorder[3]=stop=20] |   [null:pre>=length]
[null:pre>length]

class Solution {
private int in = 0, pre = 0;
public TreeNode buildTree(int[] preorder, int[] inorder) {
return buildTree(preorder, inorder, Integer.MIN_VALUE);
}
public TreeNode buildTree(int[] preorder, int[] inorder, int stop) {
if(pre >= preorder.length) return null;
if(inorder[in] == stop) {in++;return null;}
TreeNode subtree_root = new TreeNode(preorder[pre++]);//establish current subtree root
subtree_root.left=buildTree(preorder,inorder,subtree_root.val);
subtree_root.right=buildTree(preorder,inorder,stop);
return subtree_root;
}
}

leetcode106 从inorder和postorder重建二叉树：
注意通过之前题目不难发现
preorder = 从左到右的dfs
postorder_reversed = 从右到左到dfs
那么preorder+inorder 等价于reversed postorder+reversed inorder+right
所以我们继续之前解法 唯一不同是将postorder及inorder反转 然后将recursion从左到右改为从右到左 仍然是O(n) space and time

class Solution {
private int in = 0, post = 0;
public TreeNode buildTree(int[] inorder, int[] postorder) {
if(inorder.length!=postorder.length || inorder.length == 0) return null;
int[] reversed_postorder = new int[postorder.length];
int[] reversed_inorder = new int[inorder.length];
for(int i = 0; i < postorder.length;i++) {
reversed_postorder[i]=postorder[postorder.length-1-i];
reversed_inorder[i]=inorder[inorder.length-1-i];
}
return buildTree(reversed_inorder, reversed_postorder,Integer.MIN_VALUE);
}
public TreeNode buildTree(int[] inorder, int[] postorder, int stop) {
if(post >= postorder.length) return null;
if(inorder[in] == stop) {in++;return null;}
TreeNode subtree_root = new TreeNode(postorder[post++]);//establish current subtree root
subtree_root.right=buildTree(inorder,postorder,subtree_root.val);
subtree_root.left=buildTree(inorder,postorder,stop);
return subtree_root;
}
}

6. 旋转数组最小值

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。 输入一个非减排序的数组的一个旋转，输出旋转数组的最小元素。 例如数组{3,4,5,1,2}为{1,2,3,4,5}的一个旋转，该数组的最小值为1。 NOTE：给出的所有元素都大于0，若数组大小为0，请返回0。

O(N) O(1) Brute Force

class Solution {

public int findMin(int[] nums) {

int min = Integer.MAX_VALUE;

for(int i = 0; i < nums.length; i++) {

if(nums[i]<=min) min=nums[i];

}

return min;

}

}

O(logN) O(1) Binary Search

class Solution {

public int findMin(int[] nums) {

if(numsnull||nums.length0) return -1;//boundary case!

int l = 0, r = nums.length - 1, m = 0;

while(l<=r){

m = (l+r)/2;

if(nums[m] >= nums[r]) l = m+1;

else r = m;

}

return nums[m];

}

}

7. 斐波那契数列

leetcode509 现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。n<=39

方法一：O(1) O(N) Hard code all value in an array

方法二：O(2^N) O(N) NAIVE RECURSION

方法三：O(N) O(N) recursion & memoization

private Integer[] memo;


public int fib(int N) {

memo = new Integer[N + 1];

return helper(N);

}


private int helper(int N) {

if (memo[N] != null) return memo[N];

if (N <= 1) {

memo[N] = N;

return N;

}

memo[N] = helper(N - 1) + helper(N - 2);

return memo[N];

}

方法四：O(N) O(N) DP

public int fib(int N) {

if(N<=1) return N;

int[] fib = new int[N+1];

fib[0]=0;fib[1]=1;

for(int i = 2; i <= N; i++) {

fib[i]=fib[i-1]+fib[i-2];

}

return fib[N];

}

方法五：O(LOGN) O(1) 矩阵迭代 参见leetcode高票答案

8. 跳台阶

leetcode70 一次可以跳上1级台阶，也可以跳上2级。求跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

7,8,leetcode 91-decode ways, 62-unique paths均是相同的问题. 简单的dp递推 n级的解=n-1级的解+n-2级的解

class Solution {

public int climbStairs(int n) {

if(n>= 0 && n <= 2) return n;//base case

int[]f = new int[n+1];

f[0] = 0; f[1] = 1; f[2] = 2;

for (int i = 3; i <= n; i++) {

f[i] = f[i - 1] + f[i - 2];

}

return f[n];

}

}

9. 变态跳台阶

一次可以跳上1级台阶，也可以跳上2级……也可以跳上n级。求跳上一个n级的台阶总共有多少种跳法。 本题意思是可以跳任何数量为1…n的步数

n = 0: F(0) = 0;

n = 1: F(1) = 1;

n = 2: 两种跳法 第一次跳一步或者两步 F(2) = F(1) + F(0) = 2;

n = 3: 第一次跳1，2，3步 F(3) = F(2) + F(1) + F(0) = 4;

n = n: 第一次跳1，2，3，4，5，…，n步

F(n) = F(n-1)+F(n-2)+F(n-3)+…+F(n-n)=F(0)+F(1)+F(2)+…+F(n-1)

Plugin n - 1: F(n-1)=F(0)+F(1)+F(2)+…+F(n-2)

Fib(n) - Fib(n-1) = Fib(n-1), Fib(n) = 2Fib(n-1)

public class Solution {

public int JumpFloorII(int n) {

if(n >= 0 && n <= 1) return n;

int f[] = new int[n + 1];

f[0] = 0; f[1] = 1;

for (int i = 2; i < n + 1; i++) {

f[i] = 2 * f[i-1];

}

return f[n];

}

}

10. 矩形覆盖

我们可以用21的小矩形横着或者竖着去覆盖更大的矩形。请问用n个21的小矩形无重叠地覆盖一个2n的大矩形，总共有多少种方法？

仍然是一个DP问题



如图所示若右边28大矩形先被覆盖了竖着的最左边两个格子 则还有27个需要被覆盖。 若先被横着覆盖了左上角两个格子，则最左边22正方形只有1种覆盖方法，同时剩余问题规模26

F(0) = 0, F(1) = 1, F(2) = 2

N>=3 F(N)=F(N-1) + F(N-2)

public class Solution {

public int RectCover(int n) {

if(n >=0 && n <= 2) return n;

int f[] = new int[n+1];

f[0] = 0; f[1] = 1; f[2] = 2;

for(int i = 3; i < n+1;i++) {

f[i] = f[i-1] + f[i-2];

}

return f[n];

}

}

11. 二进制中1的个数

leetcode 191 Hamming weight

方法1: O(32) = O(1)

枚举每一位 每一轮右移1位得到lsb 等于1则count++ 然后rotate right 进行下一轮

public class Solution {

// you need to treat n as an unsigned value

public int hammingWeight(int n) {

int count=0;

for(int i=0;i<32;i++){

count+=(1&n);

n>>>=1;

}

return count;

}

}

注意>> signed right shift >>>unsigned right shift

方法2: O(#1) = O(1)

只数有多少个1 通过x=x&(x-1)的方式每一轮删掉最后一个1直到x=0停止计数

int hammingWeight(int n) {

int count = 0;

while (n!=0) {

n &= (n - 1);

count++;

}

return count;

}

64. 滑动窗口的最大值

sliding window of size k in array size of N 总共有n-k+1个！

滑动窗口最大值leetcode239

brute force approach: o((n-k+1)k) o(1)space

class Solution {

public int[] maxSlidingWindow(int[] nums, int k) {

if(nums.length0 || k<=0) return new int[0];

int[] result =new int[nums.length-k+1];

for(int i = 0; i < nums.length-k+1; i++){

int max=Integer.MIN_VALUE;

for(int j = i;j<i+k;j++) max = max>=nums[j]?max:nums[j];

result[i]=max;

}

return result;

}

}

Using deque : o(n) o(k)

使用双端队列来保存最大元素的index

遍历所有nums中元素

第1个while：删掉所有不在sliding window里的元素（被划过去的）

第二个while：从后往前删掉deque里所有比当前元素小的

在deque里添加当前index

只有从i=k-1起才往result里加值

class Solution {

public int[] maxSlidingWindow(int[] nums, int k) {

if(nums.length0 || k <=0) return new int[0];

int[] result = new int[nums.length - k + 1];

Deque<Integer> dq = new ArrayDeque<>();

for(int i = 0,j=0; i<nums.length;i++) {

while(!dq.isEmpty() && dq.peek()<i-k+1) dq.poll();

while(!dq.isEmpty() && nums[dq.peekLast()]<nums[i]) dq.pollLast();

dq.offer(i);

if(i >= k-1) result[j++] = nums[dq.peek()];

}

return result;

}

}
